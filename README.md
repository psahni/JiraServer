## JiraServer

This project is to utilize JIRA api's for different needs

## Setup

```
$ npm install
```

```
$ npm run start
```

## Create .env

- Copy keys from .env_example

- Generate your JIRA token from https://id.atlassian.com/manage-profile/security/api-tokens
